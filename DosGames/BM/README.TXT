Bio Menace is a DOS game and will not run from Windows.
The installation program has created a directory called \POWER14\BMENACE on
your hard drive. You will need to leave Windows and change to this directory
by typing:

CD \POWER14\BMENACE

and then type :

bmenace1

to run the game.

Full details can be found on page 17 of this month's Home PC Magazine.
